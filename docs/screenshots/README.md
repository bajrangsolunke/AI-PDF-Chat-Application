## How to take clean screenshots

1. Start backend + frontend dev servers (see project README).
2. Use Chrome DevTools device toolbar set to a 1440×900 viewport for consistency.
3. Sign up with a throwaway email, upload a small (1–3 page) sample PDF.
4. Take screenshots with your OS screenshot tool. PNG, not JPG.
5. Crop tightly — no browser chrome unless it's relevant (e.g. URL bar in the streaming shot).
6. Keep each file under ~500 KB; `pngquant --quality 65-80` if needed.

The root README references `docs/screenshots/chat.png` in its hero image — rename one of your captures to that filename, or update the README link to whichever image you pick as the hero.
